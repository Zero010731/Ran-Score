# Data release notes

This repository does not redistribute raw radiology reports, image files, local hospital reports, model checkpoints, or generated private training data.

A public data release should contain only shareable derived artifacts, such as:

- standardized finding taxonomy;
- binary finding annotations when permitted by the source dataset terms;
- report-level split manifests using permitted identifiers;
- structured audit metadata without raw report text when required;
- aggregate metrics and data dictionaries.

Users must comply with the MIMIC-CXR Data Use Agreement and any institutional restrictions for local clinical data.
